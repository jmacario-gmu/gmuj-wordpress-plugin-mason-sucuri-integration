Contributors: Jan Macario

Requires PHP: 7.0
Stable tag: 1.0

Mason WordPress plugin which provides integration with Sucuri web application firewall.

== Description ==

This plugin will store the Sucuri cache-clearing URL and allow you to clear the Sucuri cache from within the WordPress admin section.
